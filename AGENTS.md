# AGENTS.md — robocat (дайджест для внешних агентов/ревьюеров)

Ручной дайджест `CLAUDE.md` для Codex и других агентов, читающих AGENTS.md. Правило синхронизации: сначала правится `CLAUDE.md`, потом отражается здесь. Никогда наоборот.

## Проект

Discord-бот сервера Кошкокрафт (Minecraft-сообщество): Python + disnake, SQLite (aiosqlite), LLM-слой с ротацией вендоров (`bot/ai/llm.py` — единственная точка выхода в LLM API). Константы и все Discord ID — только в `bot/storage.py`. UI — disnake Components V2. Прод: VDS + systemd, автодеплой с пуша в master.

Команды: `python main.py` (запуск), `pytest -q` (тесты; сеть и .env не нужны).

## Твоя роль: независимый внешний ревьюер

Ревьюируй диффы против критериев из промпта. Ищи:

1. Нарушения критериев задачи / спеки.
2. Баги корректности с конкретным сценарием отказа (`файл:строка` + вход/состояние → неверный результат).
3. Секретность: токен Discord и ключи LLM НИКОГДА не попадают в git, логи, сообщения об ошибках.
4. Ловушку циклического импорта: `bot.utils ↔ bot.bot ↔ bot.storage` — первым всегда импортируется `bot.bot` (см. докстринг `tests/conftest.py`).
5. Гниль качества: пропущенные/ослабленные тесты, мёртвый код, захардкоженные Discord ID мимо `storage.py`.

Ранжируй по severity (blocker/major/minor). «Существенных находок нет» — валидный ответ. Файлы не изменяй.
