# Robocat
A multimodal Discord bot for a gaming community built on disnake, using **aiosqlite**, **Components V2**, and more.
The bot is used in [CATCRAFT.ru Discord](https://discord.gg/6f3FwFRJWC)!

## Features
- **Flag system** — allows attaching any text information to any Discord object: channel, user, category, etc.
- **Ticket system**
  - Contact with administration
  - Bug reports
  - In-game police appeals
- **Role selection** via Dropdown
- **Useful commands** such as FAQ, utilities, and moderation tools
- And more in the future!

## Claude Usage
Claude (Code) was involved in this project for three purposes:
- Code Review
- Security Review
- Learning

Claude did not write a single line of code or provide ready-made solutions — it only reviewed what could be improved and gave hints.

# Робокотик
Мультимодальный Discord-бот для игрового сообщества на базе disnake, использующий **aiosqlite**, **Components V2** и другое.
Бот используется в [Дискорде Кошкокрафта](https://discord.gg/6f3FwFRJWC)!

## Фичи

- **Система флажков** — позволяет добавить любую текстовую информацию на любой объект в Discord - канал, юзер, категория, т.д.
- **Тикет-система**
  - Связь с администрацией
  - Баг-репорты
  - Обращение к внутриигровой полици
- **Выбор ролей** через Dropdown
- **Полезные команды** по типу FAQ, утилит и для модерации
- И ещё много всего в будущем!

## Использование Claude
В данном проекте был задействован Claude (Code) для трёх вещей:

Claude не написал ни одной строчки кода и не выдавал готовый код, лишь смотрел что можно подправить и давал наводки.
