# Робокотик
Мультимодальный Discord-бот для игрового сообщества на базе disnake, использующий **aiosqlite**, **Components V2** и другое.
Бот используется в [Дискорде Кошкокрафта](https://discord.gg/6f3FwFRJWC)!

## Фичи

- **Система флажков** - позволяет добавить любую текстовую информацию на любой объект в Discord - канал, юзер, категория, т.д.
- **Тикет-система**
  - Связь с администрацией
  - Баг-репорты
  - Обращение к внутриигровой полици
- **Выбор ролей** через Dropdown
- **Полезные команды** по типу FAQ, утилит и для модерации

## Использование Claude
В данном проекте был задействован Claude (Code) для трёх вещей:
- Code Review
- Security Review
- Обучение

Claude не написал ни одной строчки и не выдавал готовый код, лишь смотрел что можно подправить и давал наводки.
